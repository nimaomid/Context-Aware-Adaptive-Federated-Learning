# Dataset Directory

This directory is intended to store the datasets required for reproducing the experiments reported in the CAFL framework.

Datasets used in this study:

* FLY-SMOTE
* FD-IDS

Due to dataset licensing and distribution restrictions, the datasets are not included in this repository.

Please download the datasets from their original sources and place them in this directory before running the experiments.

Expected directory structure:

data/
├── FLY-SMOTE/
└── FD-IDS/
