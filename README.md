# Context-Aware-Adaptive-Federated-Learning under Non-IID and Context Drift Conditions in IoT Networks
DOI: https://doi.org/10.5281/zenodo.20997498
Official implementation of the CAFL framework for adaptive federated learning in dynamic IoT environments.

Overview

This repository contains the implementation of the Context-Aware Adaptive Federated Learning (CAFL) framework proposed for secure and adaptive federated learning in dynamic IoT environments characterized by non-IID data distributions and context drift.
